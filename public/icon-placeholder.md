# App Icons Required

To complete the Android app setup, you need to create app icons:

1. **icon-192.png** - 192x192 pixels PNG
2. **icon-512.png** - 512x512 pixels PNG

## Icon Design Guidelines

- Use the warm orange/amber color scheme (#f2851f)
- Include a simple, recognizable symbol (e.g., road sign, steering wheel, or book)
- Ensure icons are clear and readable at small sizes
- Use transparent or solid background matching the app theme

## Quick Icon Generation

You can use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- https://www.appicon.co/

Or create them using design tools like:
- Figma
- Adobe Illustrator
- Canva

Place the generated icons in this `public/` directory.
